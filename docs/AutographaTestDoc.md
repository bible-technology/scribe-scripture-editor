## Text Translation 
### Create New Text Translation Project
- Sets:
  - Project name
  - Description
  - Abbreviation
  - New Target Language
  - Books
  - Versification Scheme
  - License
  - Imported USFM

### Edit existing Text Translation Project
  - Checks for correct:
   - Target language
   - Books
   - Versification Scheme
   - License
   
  - Edits:
   - Target language
   - Books
   - License
   - Description
   - Abbreviation

  - File
